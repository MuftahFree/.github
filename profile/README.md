## 👋 About

AG{ENT}ERPRISE is the open-source organization behind a growing set of projects focused on **agentic systems**, **Model Context Protocol (MCP)** tooling, **multi-agent prototyping**, and **AI design benchmarks**. We build the connective tissue between large language models and real-world workflows.

## 🧭 Focus Areas

- **MCP & Tool Use** — servers, managers, and clients for the Model Context Protocol.
- **Agentic Systems** — multi-agent orchestration, swarms, and autonomous task decomposition.
- **AI for Design** — benchmarks and tooling that compare and combine LLM-generated UIs.
- **Developer Experience** — open infrastructure that makes building with LLMs easier.
